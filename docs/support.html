Frequently Asked Questions

  • How do I add a server?
    Tap the + button in the top-right corner, enter a name and host, choose HTTP or TCP, then tap Save.
  • Why is my server showing as Offline?
    Make sure the host and port are correct. For HTTP checks, the server must return HTTP 200 OK. Other status codes, including redirects and 404 errors, are shown as Offline. For TCP checks, the port must be open and accepting connections.
  • How do I change the refresh interval?
    Tap the gear icon → Refresh Interval. Options range from 30 seconds to 30 minutes.
  • Why isn't the widget updating?
    Widget refresh is managed by iOS and may be throttled in Low Power Mode or when the device is idle. The widget respects the refresh interval you set in the app.
  • How do I back up or transfer my server list?
    Tap the gear icon → Export Servers. This saves a JSON file you can import on another device via Import Servers.
  • Does the app collect any personal data?
    No. All data stays on your device. No accounts, no analytics, no tracking.
